# Paradise Nursery Shopping Application

Este es el proyecto final del curso de desarrollo Front-End. Consiste en una aplicación de comercio electrónico para una tienda de plantas llamada **Paradise Nursery**.

## Características del Proyecto
- **Landing Page:** Una bienvenida atractiva con acceso directo a la tienda.
- **Listado de Productos:** Catálogo de plantas categorizado con opción de agregar al carrito.
- **Carrito de Compras:** Gestión de artículos, actualización de cantidades en tiempo real y cálculo de costos totales.
- **Estado Global:** Implementado con Redux Toolkit para una experiencia fluida.

## Tecnologías Utilizadas
- React.js
- Redux Toolkit
- Vite
- CSS3
